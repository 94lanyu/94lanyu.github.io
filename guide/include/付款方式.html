懶魚

懶魚2023年10月13日大约 2 分鐘

有 2 種付款方式

信用卡
  1. 進入 > 商城連結 > 選好商品後進入結帳頁面
  2. 填寫資料 > Email 請填入要使用的人的 Gmail 帳號
  3. 可選擇付款方式 ex.信用卡
  4. 完成後續結帳步驟
點選展開 > 結帳流程示意圖

  1. 進入 > 商城連結 > 選好商品後進入結帳頁面

  2. 填寫資料


    Email 請填入要使用的人的 Gmail 帳號
    以便透過 Gmail 發送 email 分享「可建立副本」的連結

    若聯繫的 Email 與使用人的 Google 帳號不同時，請將使用人的 Google 帳號填寫在訂單備註

  3. 可選擇付款方式 ex.信用卡

  1. 完成後續結帳步驟
APP轉帳 / 數位銀行轉帳
  1. app轉帳 ( 轉帳金額同商城品項 ) 商城連結
  2. 填入銀行代碼 - 048 ( 王道銀行 )
  3. 填入銀行帳號
    01000225022888
  4. 告知購買品項 > 依版本填入備註 - Gmail帳號 ( 要使用的人的 Gmail 帳號 )
    例 : kevin@gmail.com 要購入 → 則填入「台股訂閱版kevin」
    若備註字數超過限制則填入「台股訂閱版」後將匯款成功的畫面截圖
    並 emil 至 94lanyu@gmail.com
    ( mail 中需含轉帳末五碼截圖、Gmail、轉帳日期、購買品項 )
